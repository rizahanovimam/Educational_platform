# Образовательная платформа с ИИ-наставником

## 🎯 Что это за платформа

**Образовательная платформа с ИИ-наставником** — это веб-интерфейс, который показывает, как может быть организовано обучение программированию в диалоге с искусственным интеллектом.

**Главное отличие от обычных курсов:** не ты ищешь материал, а ИИ-наставник ведёт тебя по индивидуальной программе, объясняет сложные темы несколько раз по-разному и всегда готов ответить на вопросы. Платформа помогает понять, с чего начать, какие темы учить и когда переходить к следующему материалу.

**Формат обучения:** беседа с ИИ-преподавателем в чате + практика во встроенной песочнице. Всё в одном окне — не нужно переключаться между учебником, компилятором и поисковиком.

---

## ⚠️ Важное примечание

**Данный проект является концептуальным прототипом (макетом) образовательной платформы.**

Прототип демонстрирует внешний вид, структуру интерфейса и логику организации образовательного процесса. Полноценного функционирования всех заявленных возможностей в текущей версии нет — большинство интерактивных элементов представлены в виде имитации, показывающей, как они будут выглядеть и работать в готовом продукте.

## ✅ Что уже работает

В проекте подразумевается, что пользователь уже прошёл диагностику и один курс «Указатели в C++: от адресной арифметики до умных указателей» добавлен в его личный кабинет. Это сделано для наглядной демонстрации интерфейса: отображается прогресс обучения (19%), количество модулей и задач, а также доступны кнопки «Продолжить обучение» и «Посмотреть прогресс».

В текущей версии реализованы следующие полностью функционирующие модули:

| Модуль | Описание | Статус |
|--------|----------|--------|
| **Навигация** | Фиксированное меню, адаптивное мобильное меню (гамбургер), переходы между страницами | Работает |
| **Адаптивная вёрстка** | Корректное отображение на компьютерах, планшетах и мобильных устройствах | Работает |
| **Заметки** | Добавление и удаление текстовых заметок. Данные сохраняются в localStorage и не исчезают после перезагрузки страницы | Работает |
| **Словарь терминов** | Добавление новых терминов с названием и определением. При клике на термин отображается его определение. Удаление — правой кнопкой мыши. Данные сохраняются в localStorage | Работает |
| **Симулятор переводчика** | Имитация работы переводчика: вводишь текст, нажимаешь «Перевести» — система выводит введённый текст с пометкой «[Перевод]» | Работает |

## ❌ Что пока не работает (требует доработки)

Следующие функции представлены в виде имитации или статичного макета. Для их полноценной работы требуется подключение внешних API и серверной логики.

| Функция | Текущее состояние | 
|---------|-------------------|
| **Само ядро программы (обучение)** | Отсутствует — нет системы управления курсами, нет базы знаний, нет логики формирования программы обучения. Пользователь не может реально изучать материал | 
| **Чат с ИИ-консультантом (диагностика)** | Чат статичен — отображается только внешний вид интерфейса, отправить сообщение или получить ответ невозможно |
| **Чат с ИИ-преподавателем (обучение)** | Отображается интерфейс чата, но вести диалог нельзя. При вводе сообщения ничего не происходит | API нейросети + база знаний |
| **Песочница (редактор кода)** | В редактор можно вводить и редактировать код, но кнопки «Запустить» и «Проверить» не выполняют компиляцию и не анализируют код | 
| **Переводчик (полноценный)** | Сейчас только симулятор — выводит введённый текст с пометкой «[Перевод]», реального перевода нет | 
| **Регистрация и авторизация** | Отсутствует |
| **Сохранение прогресса между сессиями** | Прогресс и заметки сохраняются в localStorage, но привязаны к конкретному браузеру и устройству |
